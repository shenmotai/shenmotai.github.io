数据库的概念

简单地说数据库是按一定规则存储并能够修改成千上万条数据的厂库。

MySQL数据库的下载

首先去MySQL官网下载（全英文真的好难），或者去b站找相关视频下载并学习。

MySQL数据库的打开方式

首先确保MySQL服务器是打开状态
可以用cmd窗口输入net stat ＋服务器名字 打开服务器

1、 可以在开始菜单搜索MySQL专用黑窗口，打开后直接输入安装时设置的密码即可使用

2、使用cmd窗口输入mysql -u root -p＋密码（p和密码之间没有空格）
